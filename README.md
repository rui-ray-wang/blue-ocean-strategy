# Blue Ocean Strategy --- Strategy Canvas Activity

This site is designed for an **in-class activity** on building a
**Strategy Canvas**, based on *Blue Ocean Strategy* concepts.

## 🌐 Live Site

https://rui-ray-wang.github.io/blue-ocean-strategy

## 📘 Purpose

The website supports classroom learning by helping students practice
visualizing competitive strategy using a Strategy Canvas.

## ✏️ What You'll Find

-   An **example Strategy Canvas** demonstrating how the framework
    works\
-   A **blank canvas template** for students to build their own analysis
    during class activities

## 🎯 Intended Use

This tool is created for instructional use in strategy or management
courses, enabling students to: 

- Identify key competitive factors 
- Compare industry players 
- Visualize strategic positioning 
- Develop blue ocean insights through hands-on practice
